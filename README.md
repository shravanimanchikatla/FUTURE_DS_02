# FUTURE_DS_02
Customer Churn Analysis Dashboard
Project Overview:
This project presents an interactive Customer Churn Analysis Dashboard . The dashboard helps analyze customer behavior, identify churn patterns, evaluate revenue, and understand factors influencing customer retention.
The objective is to transform raw customer data into meaningful business insights through interactive visualizations and KPIs.
Tools Used:
-Microsoft Power BI
-Microsoft Excel
Business Insights:
-The dataset contains approximately 7,032 customers.
-The average monthly charge is 64.80, indicating the typical monthly spend per customer.
-Most customers are retained, while a smaller portion has churned.
Visualizations:
The dashboard consists of the following visualizations to analyze customer churn and business performance:
-KPI Cards
Total Customers
Total Revenue (Total Charges)
Average Monthly Charges
-Donut Chart
Customer Churn Distribution (Churned vs. Retained Customers)
-Clustered Column Chart
Average Monthly Charges by Churn Status
-Line Chart
Customer Tenure vs. Churn
-Stacked Column Chart
Customer Distribution by Gender and Churn Status
-Column Chart
Revenue Generated by Contract Type
-Interactive Slicer
Filter dashboard by Gender
-Hierarchy Slicer
Filter data using Contract Type, Internet Service, and Payment Method
